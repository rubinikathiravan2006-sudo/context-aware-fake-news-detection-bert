# context-aware-fake-news-detection-bert
AI-based fake news detection system using BERT, Graph Neural Networks (GNN), and Softmax classifier with NLP techniques to classify news as Real or Fake using contextual understanding and confidence score prediction.

## Dataset
- WELFake Dataset

## Technologies Used
- Python
- BERT
- Graph Neural Networks (GNN)
- Softmax Classifier
- Transformers
- Gradio
- Scikit-learn

## Features
- Context-aware fake news detection
- Real-time prediction
- Confidence score display
- NLP preprocessing
- User-friendly interface
